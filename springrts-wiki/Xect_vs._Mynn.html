The Xect vs. Mynn Mod, aims to be a completely free (as in GNU GPL) data package for the Spring engine.

History

This mod was initialy created by Chris Jorgensen http://www.xenoclone.com/ for the original Total Annihilation engine. For the Spring engine Chris Jorgensen decided to release his work under the GNU GPL so others could port it. Michilus Nimbus made the first port that did not crash on startup. FoeOfTheBee has made it somewhat playable, and probably GPL.
